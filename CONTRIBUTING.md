# Contributing to taiwan-weight-loss-plateau-science

## 本資料庫的核心立場

減重停滯期是一個高度商業化的議題，充斥「突破停滯的秘密」、「代謝提升食物」等誇大宣稱。本資料庫：

1. 以**生理機制**解釋停滯期，而非歸咎於「意志力不夠」
2. **誠實呈現效果幅度**——代謝適應真實存在，但幅度有限（非「吃什麼都會胖」）
3. 突破策略需有 **RCT 支持**，而非只靠理論推論

## 接受的貢獻

- 代謝適應的新 RCT 或 Meta-analysis（需附 PMID）
- 荷爾蒙追蹤研究（需人體研究，動物研究僅補充機制說明）
- 台灣減重行為研究（NHRI、學術機構）
- 無效策略的研究（確認哪些方法不管用同樣重要）

## 不接受的貢獻

- 特定商業減重計畫或產品的推薦
- 「代謝提升食物」（辣椒、咖啡、茶）的誇大效果宣稱（若要納入需精確呈現效果幅度）
- 動物實驗作為人體策略的主要依據
- 傳統偏方（薑茶、特定草藥加速代謝）無 RCT 支持的聲明

## 證據分級標準

- **A 級**：至少 1 個高品質 RCT 或 ≥ 3 個 RCT 的 Meta-analysis（人體研究）
- **B 級**：觀察性研究、小樣本 RCT、或機制有合理人體支持
- **C 級**：初步/探索性人體研究，缺乏足夠重複驗證
- **D 級**：個案報告、動物研究、試管研究

## 重要：不允許選擇性引用

每個突破策略的貢獻需包含：
- `honest_assessment`（誠實評估）：效果幅度的實際大小
- 已知的局限性和反面研究

---

本資料庫以科學教育為目的，不構成醫療建議。減重停滯期持續 6–8 週以上建議就醫排除甲狀腺疾病等代謝問題。
